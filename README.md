# 📋 Fluxo
> **Looking for the process?** [Check out my portfolio here](https://matildaerenius.github.io/portfolio/projects/fluxo.html) 🚀

Fluxo is a task tracking application where users can create and organize task lists, set priorities and due dates, track completion progress, and manage their workflow. Structured as a monorepo, the project features a Spring Boot backend alongside a modern Android frontend built with Jetpack Compose and Clean Architecture principles.
